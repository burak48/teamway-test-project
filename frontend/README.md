# Frontend Installation

<br>

## Project setup
```
yarn install
```

<br>

## Compiles and hot-reloads for development
```
yarn serve
```

<br>

## Compiles and minifies for production
```
yarn build
```

<br>

## Run your unit tests
```
yarn test:unit
```

<br>

## Lints and fixes files
```
yarn lint
```

<br>
